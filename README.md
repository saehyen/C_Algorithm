#### 문자열 내림차순으로 배치하기
- compare 함수 만들기
- c언어 내부 quick sort가 존재함
- qsort (void *base, size_t nel, size_t width, int (*compare)(const void *, const void *);
- base: 정렬하고자 하는 배열의 포인터
- nel: 배열의 각 원소들의 총 수
- width: 배열에서 원소 하나의 크기입니다.
- *compare: 비교를 수행할 함수 포인터
- 오름차순시 compare함수에서 클 때 1 작을때 -1, 내림차순은 반대로

- 선택정렬도 가능
- 문자마다의 비교 = 파이썬과 같음
- swap함수를 define하고 사용할 것
